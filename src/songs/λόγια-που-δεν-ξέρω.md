1.Για τη ζωή που μου χάρισες\
Θα Σε υπηρετώ,\
Για κάθε μάχη που κέρδισες\
Σε δοξολογώ.

2. Στης κάθε αυγής το χαμόγελο\
Θα Σε προσκυνώ,\
Φτωχός και πένης στον κόσμο αυτόν\
Θα Σ’ ακολουθώ.

*Κύριε ελέησέ με,*\
*Μάθε μου να ζω,*\
*Πώς να Σε υμνήσω*\
*Και πώς να σταθώ.*\
*Τι να Σου προσφέρω,*\
*Τι σκοπό να βρω,*\
*Λόγια που δεν ξέρω*\
*Είπες στο σταυρό.*
